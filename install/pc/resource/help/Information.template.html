An Enterprise Data Protection Enforcer is currently installed on this PC. For information about this Enforcer, right-click the red Data Protection icon in your system tray.

Select About Enterprise Data Protection to view the following:

  • The version of the installed software
  • The current status of the Enforcer (either Running, or Not Running)
  • The date and time when the most recent new policy was deployed on this PC by the central server

Select View Notifications List for a list of all instances of any policy being enforced on this PC, since the last time it was rebooted. Enforcement refers to any time a policy is triggered by some user action. Note that policies do not necessarily prevent an action; they can also allow an action while displaying a reminder message, or performing some automated task, or simply logging the action for auditing purposes. The View Notifications List window shows:

  • The date and time of the enforcement
  • The event that triggered the enforcement
  • The name of the file involved
  • The message that was displayed
