An Enterprise Data Protection Enforcer is currently running on this PC, to help you as you work by displaying notification reminders about how to handle sensitive data, preventing you from inadvertently exposing or losing such data, and automating required workflows to save you time and effort.

Our IT administrators have deployed a number of information access and use policies to make it easier for you to protect the content of sensitive documents. Whenever you trigger one of these policies by performing some action on a file or other data resource (such as SharePoint content), the Desktop Enforcer displays a notification message either as a pop-up balloon in the lower right corner of the screen, or as a standard Windows error message.

Unless you have the administrative password, you cannot turn the enforcer off; it is designed to enforce all currently deployed policies continuously. However, you can turn off the notification balloons if you like; just right-click on the red Data Protection icon in your system tray, and click Display Notifications to uncheck it. Bear in mind that even when the display is turned off, you may still see instances when you are unexpectedly prevented from opening documents on a file server, or from performing certain actions with documents: that is the Desktop Enforcer at work.
