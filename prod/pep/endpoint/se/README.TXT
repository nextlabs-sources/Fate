Module               Description
----------------------------------------------------------------------------
nl_crypto            Kernel-mode cryptography library.  Implements AES-128.

nl_sysenc            System Encryption management tool.  Directory and per
                     file encryption.

nl_sysenc_lib        System Encrpytion library provided to retrieve info
                     about encrypted files.  Management of directories and
                     files is also provided.

nl_SysEncryption     Kernel-mode file system filter driver with user-mode
                     Policy Controller plug-in.

nlseRecovery         System Encryption recovery tool.
